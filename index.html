Budchat

A real time minimalist chat room app that hearkens back to the days of old instant messengers such as AOL Messenger, and MSN Messenger made as my two week solo project in the final weeks of the Emerging Digital Academy Jolitz cohort. Users can make profiles, talk to other users, and create new rooms to talk to other users in. They may also add other users to a buddy list so they may be invited to private rooms.
Built with React

A CRUDdy To-Do List

This is an app made as an assignment during tier two of the Emerging Digital Academy. The user may add, and delete tasks as they see fit. The user can see on the table whether they have done a task or not, and if they had done a task, when it was done. The user may also sort the task table to see done tasks first instead of not done tasks.

Pizza Typer

A simple game where the main objective is to create as many pizzas by typing their ingredients as possible before either the time runs out, or mistakes are made. It features 2d graphics, and sound. Made almost entirely with plain Javascript. Kid friendly too.
